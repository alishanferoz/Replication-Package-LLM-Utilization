# Software Development & LLM Survey Analysis Suite

This repository contains a collection of Python scripts used to process, analyze, and visualize survey data regarding the use of Large Language Models (LLMs) in the Software Development Lifecycle (SDLC).
---

## 📋 Prerequisites

Ensure you have Python 3.8+ installed. You can install all required libraries using pip:

```bash
pip install pandas numpy matplotlib seaborn wordcloud openpyxl scipy google-generativeai
```

---

## 🛠 Script Breakdown & Generated Charts

### 1. General Demographics & SDLC Overview
File:analyze_survey2latest.py

This script provides the foundational analysis of who responded and how they use LLMs generally.

- How to run: python analyze_survey2latest.py
- Charts Generated:

    - chart_A_experience.png: Years of experience distribution.

    - chart_B_domain_cloud.png: Word cloud of software development domains.

    - chart_C_continent_heatmap.png: Geographic distribution of respondents.

    - chart_D_phases.png: LLM adoption across Requirements, Design, Coding, and Testing.

    - chart_I_diagrams.png: Specific design diagrams generated by LLMs.

    - chart_J_scenarios.png: Types of test scenarios generated.

    - chart_J_codegen.png: Specific code generation tasks performed.

    - chart_K_future.png: Pie chart on whether LLMs will replace developers.

### 2. Dual-Axis Usage vs. Benefit Analysis

File: analyze_survey_latest.py

This script creates specialized charts that compare Frequency of Use (Usage) against Perceived Value (Benefit) for the same tasks.
- How to run: python analyze_survey_latest.py

- Charts Generated:

    - chart_E_req_combined.png: Usage/Benefit for Requirement Gathering tasks.

    - chart_F_design_combined.png: Usage/Benefit for Architecture & Design tasks.

    - chart_F_dev_combined.png: Usage/Benefit for Coding/Development tasks.

    - chart_F_test_combined.png: Usage/Benefit for Software Testing tasks.
  
### 3. Professional Likert & Limitation Analysis
File: analyze_survey2_latest.py

This script uses high-resolution settings and manual label mapping to create polished charts for qualitative "Limitations" and "Access Methods."

- How to run: python analyze_survey2_latest.py
- Charts Generated:

    - chart_F_limits.png: Frequency of limitations (e.g., Syntax errors, Privacy concerns, Hallucinations).

    - chart_F_access.png: Popularity of access methods (IDE Plugins, Web Chat, CLI).
  
    - chart_H_benefit_int.png: Percentage breakdown of benefits from LLM integration.
### 4. Statistical Significance (Fisher's Exact Test)
File: fischer_exact_test.py

Instead of images, this script performs statistical validation of the survey results to find significant patterns.

- How to run: python fischer_exact_test.py

- Output:

    - Significance test analysis_Output.xlsx: A multi-sheet Excel file containing Contingency Tables, Odds Ratios, P-values, and color-coded Significance Matrices.

## ⚙️ Customization
- Manual Label Mapping

    - If survey questions change, you can update the MANUAL_MAP dictionary in analyze_survey2_latest.py to ensure the chart labels remain concise and professional.
- Visual Style

    - You can modify colors by changing the hex codes in the COLORS_LIKERT_5 or COLORS_BENEFIT_5 variables found at the top of each script.
